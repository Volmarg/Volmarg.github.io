Writing module backend logic

There is simple logic to follow here. Put every controller, entity, form and repository into corresponding module folder

Controller

  • Each controller for given module should go to
    • src/Controller/Modules/<moduleName>

Entity

  • Each Entity for given module should go to
    • src/Entity/Modules/<moduleEntity>

Repository

  • Each Repository for given module should go to
    • src/Repository/Modules/Modules/<moduleEntity>

Form

  • Each Form for given module should go to
    • src/Form/Modules/Modules/<moduleEntity>
